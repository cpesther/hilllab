# Hill Lab Module
The hilllab module contains all Python and C# code written for use in the David Hill lab. This codebase was written and designed by [Christopher Esther](mailto:christopher_esther@med.unc.edu) with some exceptions noted below. 

The ptmr submodule contains Python translations of MATLAB scripts originally written by Matt Markovetz and David Hill with additional statistics and visualization code by Christopher Esther. 

The magnets submodule contains some Python code written by Mahmud Riahan.
